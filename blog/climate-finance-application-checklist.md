# Climate Finance Application Checklist
## A Step-by-Step Guide for Kenyan Applicants

**Download this checklist and use it to track your application progress**

---

## FOR ALL FUNDING SOURCES (Universal Requirements)

- [ ] **Legal Registration**: Verify your organization's legal registration is current
- [ ] **Tax Compliance**: Obtain tax compliance certificate from Kenya Revenue Authority (KRA)
- [ ] **Banking**: Open/verify dedicated bank account with proper governance
- [ ] **Community Assessment**: Conduct participatory community needs assessment
- [ ] **Baseline Data**: Document baseline data (climate vulnerability, current conditions)
- [ ] **Project Concept**: Develop project concept with clear climate rationale
- [ ] **Co-financing**: Identify potential co-financing sources
- [ ] **Organizational Documents**: Prepare organizational documents (registration, governance, financial statements)

---

## ADDITIONAL REQUIREMENTS FOR GREEN CLIMATE FUND (GCF)

**Timeline: 24-48 months | Funding Range: $10M - $250M**

### Pre-Application (3-6 months)
- [ ] Review [Kenya's NDC](https://unfccc.int/NDCREG) and ensure project alignment
- [ ] Contact National Treasury (NDA) for guidance
  - Email: climatechange@treasury.go.ke
- [ ] Identify and partner with accredited entity:
  - [ ] NEMA (National Implementing Entity)
  - [ ] FAO (for agricultural projects)
  - [ ] Other accredited entity

### Documentation Phase (6-12 months)
- [ ] Commission full Environmental and Social Impact Assessment (ESIA) if required
- [ ] Develop comprehensive gender action plan (mandatory)
- [ ] Prepare detailed technical feasibility studies
- [ ] Create stakeholder engagement plan
- [ ] Document paradigm shift potential of your project
- [ ] Prepare co-financing documentation (30-50% required)

### Submission Phase (1-2 months)
- [ ] Submit concept note through accredited entity
- [ ] Attend NDA stakeholder consultations
- [ ] Obtain NDA no-objection letter
- [ ] Address any feedback from National Treasury

### Full Proposal Phase (6-12 months)
- [ ] Develop full proposal with accredited entity
- [ ] Complete all GCF Environmental and Social Safeguards documentation
- [ ] Finalize budget with detailed justifications
- [ ] Prepare monitoring and evaluation framework
- [ ] Submit full proposal to GCF Secretariat
- [ ] Respond to Independent Technical Advisory Panel (ITAP) feedback

### Approval Phase
- [ ] Track GCF Board meeting schedule (Quarterly: Feb, May, Aug, Nov)
- [ ] Prepare for potential Board presentation
- [ ] Await Board decision

## ADDITIONAL REQUIREMENTS FOR COUNTY CLIMATE CHANGE FUNDS (CCCF)

**Timeline: 3-6 months | Funding Range: $5K - $1M**

### Community Engagement Phase (1 month)
- [ ] Identify your Ward Adaptation Planning Committee (WAPC)
  - Contact: County Director of Environment
- [ ] Participate in ward climate vulnerability assessment
- [ ] Present project idea at community meetings
- [ ] Ensure project is prioritized by community members
- [ ] Document community support and participation

### Proposal Development (1-2 months)
- [ ] Work with WAPC to develop proposal
- [ ] Confirm county has enacted climate change legislation
- [ ] Secure community co-financing commitment (can be in-kind)
- [ ] Complete technical viability assessment
- [ ] Prepare simple budget (focus on public goods)
- [ ] Document how project benefits the ward community

### County Approval Process (1-2 months)
- [ ] Submit through WAPC to County Climate Change Unit
- [ ] Address technical feedback from county environment office
- [ ] Attend County Climate Change Steering Committee presentation
- [ ] Participate in public participation notice process
- [ ] Obtain County Executive Committee approval

### Implementation Readiness
- [ ] Establish project management committee with community representation
- [ ] Set up community oversight mechanisms
- [ ] Prepare for immediate implementation upon MOU signing

## ADDITIONAL REQUIREMENTS FOR KCIC (Kenya Climate Innovation Center)

**Timeline: 3-6 months | Funding Range: $10K - $500K**

### Business Preparation (1-2 months)
- [ ] Complete business model canvas
- [ ] Prepare comprehensive market analysis
- [ ] Document competitive landscape
- [ ] Develop financial projections (3-5 years)
- [ ] Calculate climate impact metrics
- [ ] Define scalability pathway

### Application Package
- [ ] Create pitch deck (10-15 slides) covering:
  - [ ] Problem statement
  - [ ] Climate innovation solution
  - [ ] Market opportunity
  - [ ] Business model
  - [ ] Team qualifications
  - [ ] Financial projections
  - [ ] Climate impact
  - [ ] Funding request and use of funds

- [ ] Prepare detailed business plan
- [ ] Document team profiles and experience
- [ ] Identify co-financing/matching funds sources
- [ ] Prepare financial statements (if existing business)

### Online Application (2-4 weeks)
- [ ] Submit online application via [KCIC portal](https://www.kenyacic.org)
- [ ] Upload all required documents
- [ ] Complete application form accurately
- [ ] Submit before deadline (if call-specific)

### Due Diligence Phase (1-2 months)
- [ ] Prepare for site visit
- [ ] Organize customer/market validation evidence
- [ ] Prepare reference contacts
- [ ] Update financial information
- [ ] Document any prototype/proof of concept

### Investment Committee Phase (2-4 weeks)
- [ ] Practice investment committee pitch (10-15 minutes)
- [ ] Prepare responses to potential questions:
  - [ ] Market validation
  - [ ] Competitive advantage
  - [ ] Team capability
  - [ ] Financial sustainability
  - [ ] Climate impact measurement
  - [ ] Scaling strategy
- [ ] Attend pitch session
- [ ] Respond to committee feedback

## ADDITIONAL REQUIREMENTS FOR ADAPTATION FUND

**Timeline: 12-18 months | Funding Range: $1M - $10M**

### Pre-Application (2-4 months)
- [ ] Contact NEMA (National Implementing Entity)
  - Website: [www.nema.go.ke](https://www.nema.go.ke)
- [ ] Develop project concept note
- [ ] Ensure alignment with Kenya's adaptation priorities
- [ ] Identify co-financing sources (10-20%)

### Proposal Development (3-6 months)
- [ ] Conduct community-based vulnerability assessment
- [ ] Develop full project proposal (minimum 10,000 beneficiaries)
- [ ] Prepare environmental and social assessment
- [ ] Create gender-responsive implementation plan
- [ ] Develop monitoring and evaluation framework
- [ ] Complete stakeholder consultation process

### Submission and Review (4-6 weeks)
- [ ] Submit through NEMA to Adaptation Fund
- [ ] Respond to Secretariat technical review comments
- [ ] Address any compliance issues

### Board Approval
- [ ] Track Adaptation Fund Board meeting schedule (twice per year)
- [ ] Prepare for Board technical review
- [ ] Await Board decision

---

## ADDITIONAL REQUIREMENTS FOR GLOBAL GREENGRANTS FUND

**Timeline: 2-4 months | Funding Range: $500 - $5K**

### Simple Application Process
- [ ] Describe project in simple terms (no jargon)
- [ ] Explain environmental justice angle
- [ ] Document community leadership and ownership
- [ ] Provide references from community members
- [ ] Submit brief budget (very simple)
- [ ] No formal feasibility study required
- [ ] No formal environmental assessment required
- [ ] Tax compliance helpful but not mandatory

## COMMON DOCUMENTS NEEDED (Keep These Ready)

### Organizational Documents
- [ ] Certificate of registration/incorporation (certified copy)
- [ ] Organization constitution/bylaws
- [ ] Tax compliance certificate (KRA - updated)
- [ ] Bank account details and statements (3-6 months)
- [ ] Audited financial statements (if required by fund)
- [ ] Board/committee resolution authorizing application
- [ ] Organizational profile/track record

### Project Documents
- [ ] Project concept note (2-5 pages)
- [ ] Detailed project proposal
- [ ] Budget with justifications
- [ ] Logical framework/theory of change
- [ ] Work plan/timeline
- [ ] Monitoring and evaluation plan
- [ ] Risk assessment and mitigation plan
- [ ] Sustainability plan

### Community/Beneficiary Documents
- [ ] Community needs assessment report
- [ ] Stakeholder mapping
- [ ] Letters of support from community leaders
- [ ] Evidence of community consultations (attendance lists, photos)
- [ ] Baseline data on beneficiaries

### Technical Documents (as required)
- [ ] Environmental Impact Assessment or screening
- [ ] Social Impact Assessment
- [ ] Technical feasibility study
- [ ] Market analysis (for business-oriented projects)
- [ ] Climate vulnerability assessment
- [ ] Gender analysis and action plan

### References and Contacts
- [ ] Organizational references (3-5)
- [ ] Partner organization agreements
- [ ] Government approvals/no-objection letters
- [ ] Co-financing commitment letters

## SUCCESS TIPS

### Before You Apply
✓ Start with smaller grants to build track record
✓ Understand the specific funder's priorities
✓ Visit funded projects to learn from successful applicants
✓ Build relationships with fund managers before applying
✓ Join climate finance training workshops

### During Application
✓ Be specific about climate vulnerability and how you'll address it
✓ Show genuine community ownership and participation
✓ Demonstrate sustainability beyond the funding period
✓ Include realistic, well-justified budgets
✓ Highlight innovation and replication potential
✓ Address gender and social inclusion explicitly
✓ Provide credible baseline data and impact metrics

### Common Mistakes to Avoid
✗ Generic climate rationale without local specifics
✗ Overly ambitious scope without proven capacity
✗ Weak community engagement (consultation ≠ participation)
✗ Inadequate budget justifications
✗ Missing gender analysis
✗ No baseline data for measuring impact
✗ Unclear sustainability plan
✗ Missing required documents
